# ManeKelsa

ManeKelsa is an Android application developed using Kotlin and Firebase as part of the internship project titled **“Android Application Development using Generative AI.”**

## Project Overview
The application is designed to connect local workers with users searching for nearby services. It acts as a digital worker directory where users can easily discover skilled workers based on profession and availability.

## Features
- Worker registration and profile management
- Nearby worker directory
- Real-time data using Firebase Firestore
- Authentication using Firebase
- Job management interface
- User profile management
- Responsive Android UI
- Clean and user-friendly design

## Technologies Used
- Kotlin
- Android Studio
- Firebase Firestore
- Firebase Authentication
- XML Layouts
- RecyclerView
- MVVM Architecture

## Modules
- Login & Authentication
- Worker Directory
- My Jobs
- Profile Management
